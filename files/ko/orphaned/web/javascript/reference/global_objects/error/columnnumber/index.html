{{JSRef}} {{non-standard_header}}

columnNumber 속성은 이 오류가 발생한 파일의 행의 열 번호를 포함합니다.

예제

columnNumber 사용하기

var e = new Error('Could not parse input');
throw e;
console.log(e.columnNumber) // 0

명세

표준의 일부가 아닙니다.
